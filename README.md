# Calculator
This is Basic web-based calculator that performs simple arithmetic operations like addition , subtraction , multiplication , and division.
It is built using HTML , CSS and JavaScript

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Clear function
- Responsive design

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
